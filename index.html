Deskripsi Sistem

Sistem ini menyediakan halaman informasi kegiatan mahasiswa serta API untuk menghitung estimasi biaya konsumsi. Perhitungan dilakukan berdasarkan parameter jumlah peserta dan jenis konsumsi yang dipilih.

Layanan Azure yang Digunakan

Komponen cloud yang digunakan selama praktikum

S

Static Web Apps

Hosting halaman informasi kegiatan secara statis.

F

Azure Functions

API estimasi biaya berbasis HTTP Trigger.

A

App Service

Menjalankan aplikasi web dinamis dan REST API.

C

Container Apps

Menjalankan aplikasi berbasis container image.

Dokumentasi API Estimasi Biaya

API menerima parameter jumlah dan jenis konsumsi

Endpoint

/api/EstimasiBiaya?jumlah=40&jenis=snack

Parameter

jumlah : jumlah peserta kegiatan
jenis  : snack | makan | lengkap

Contoh Response JSON

{
  "jumlah": 40,
  "jenis": "snack",
  "hargaPerPeserta": 15000,
  "total": 600000,
  "timestamp": "2026-05-06T10:00:00.000Z"
}
